IPSCANNER-1.0
ИНСТРУКЦИЯ
1. Заполните форму ниже. Все поля обязательны.
2. Введите начальный IP адрес и диапазон. Допустимые значения: 0-255.
3. Выберите количество потоков.
4. Нажмите кнопку СТАРТ.
5. Программа найдет все SSL сертификаты по заданным адресам и найдет в них все доменные имена.
6. Список найденных доменных имен будет сохранен в файле result.txt
